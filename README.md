### 📊 Interactive Scanning Matrix
- Drag & drop signals  
- Custom axes (Impact, Probability, etc.)  
- Real-time repositioning  

---

### 🤖 AI Strategic Analysis
- Pattern detection  
- Cluster identification  
- Executive-level insights  
- Actionable recommendations  

---

### 🧠 Signal Intelligence System

- 🟢 Emerging  
- 🔵 Watchlist  
- 🟡 Escalating  
- 🔴 Critical

❌ Groupthink → Visualize divergence
</details>

🧪 Power Feature: Spread Overlap

<details> <summary>Click to expand</summary>

When signals overlap:

✔️ Enable Spread Overlap
✔️ Nodes visually separate
✔️ Data remains unchanged
</details>
